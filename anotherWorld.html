Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. innerHtml returns all text, including html tags, that is contained by an element. 2.innerText returns all text contained by an element and all its child elements. 3. textContents is all text contained by an element and all its children that are for formatting purposes only.

Question-3: What is event delegation in the DOM?

Event delegation in JavaScript is a pattern that efficiently handles events. Events can be added to a parent element instead of adding to every single element. It refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in the DOM (Document Object Model) event propagation mechanism where an event that occurs on a nested element first triggers on that element and then propagates (or "bubbles up") to its parent elements and then further up to the root (document).
